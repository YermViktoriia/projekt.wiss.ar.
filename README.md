# Titanic Datenanalyse Projekt

## Beschreibung
Dieses Projekt wurde im Rahmen einer Gruppenarbeit durchgeführt. Ziel war die Analyse des Titanic-Datensatzes unter Verwendung von Git und GitHub sowie die Anwendung von R für Datenaufbereitung, Funktionsentwicklung und Datenanalyse.

## Inhalt des Repositories

### Datensätze
- `titanic.csv` – Originaldatensatz  
- `titanic_clean.csv` – Aufbereiteter Datensatz für die Analyse  

### R-Skripte
- `01_preprocessing.R` – Datenaufbereitung und Feature Engineering  
- `02_functions_main.R` – Funktionen für deskriptive Analyse und Visualisierung  
- `03_functions_helpers.R` – Hilfsfunktionen für Typüberprüfung  
- `04_analysis.R` – Durchführung der Analyse mit den erstellten Funktionen  

### Bericht
- `Titanic_Analyse_Report.pdf` – Abschlussbericht mit Beschreibung der Ergebnisse  

## Analyseziel
Untersuchung von Zusammenhängen zwischen:
- Geschlecht
- Reiseklasse
- Alter
- Ticketpreis
- Überlebenswahrscheinlichkeit

## Verwendete Methoden
- Deskriptive Statistik
- Bivariate Analyse
- Datenvisualisierung

## Technologien
- R
- Git
- GitHub
- LaTeX

## Hinweise
Alle Skripte wurden so erstellt, dass sie ohne Fehler ausführbar sind. Die Datenaufbereitung erfolgt im Preprocessing-Skript, anschließend wird der bereinigte Datensatz für die Analyse verwendet.
